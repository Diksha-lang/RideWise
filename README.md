# RideWise

# Overview

RideWise is a simple console-based ride sharing application developed in Java.

The system allows users to register riders and drivers, book rides, calculate fares, and track ride status.

This project was developed to practice Java OOP concepts, SOLID principles, and the Strategy Design Pattern.
